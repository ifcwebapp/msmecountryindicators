Description Note on the MSME Country Indicators 2014
Please read the “Description Note on the MSME Country Indicators 2014” along with the country specific comments in the Micro, Small, and Medium Enterprise Country Indicators (MSME-CI) MS Excel workbook before using the data. MSME-CI presents secondary data collected by various institutions (statistical institutes, ministries, international organizations, small business promotion agencies, research institutions and others) using different methods (survey, census and others). Description Note on the MSME Country Indicators
The description note provides an overview of the terms and use of the data, the data collection process and scope, notes on selected variables, an overview of the structure of the database, and the steps taken to clean the data for the analysis undertaken. It also provides additional notes on the data, methodological issues in regards to single and monetary values, and the determination of the most widely used MSME definition by country.
