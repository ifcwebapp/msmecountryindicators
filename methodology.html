Methodology Note on the MSME Country Indicators
The methodology note provides an overview of the terms and use of the data, the data collection process and scope, notes on selected variables, an overview of the structure of the database, and the steps taken to clean the data for the analysis undertaken. It also provides additional notes on the data, methodological issues in regards to single and monetary values, and the determination of the most widely used MSME definition by country. The last section provides an overview of the quality control steps that were undertaken and some lessons learned from the data collections process. Read full document
